Изучил тему роутинга в фастапи, разделение на абстракции. Для объявления роутинга, требуется импортировать класс из fastapi под названием APIRouter.
Далее, создаем экземпляр класса, условный router = APIRouter(), из объявленного экземпляра класса вызываем методы get, post, put и т.д последством router.get('example/posts')
Импортируем наш роутер в исполняемый скрипт и с помощью метода из экзмпл класса app импортируем include_router: app.include_router(router)
